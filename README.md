## Project Intro/Objective
The purpose of this project is to build an AI-powered system to be able to detect Sarcasm or Fake News from text content, to be more specific news headlines.

### Methods Used
* Deep Learning
* NLP
* Word2Vec
* RNN-LSTM

### Technologies
* Python
* Pandas, google colab
